# B2B Trade Portal

A full-stack web application that connects buyers and sellers for bulk trading without middlemen.

## 🚀 Features

- User authentication (Buyer & Seller roles)
- Product listing and management
- Order placement and tracking
- Admin dashboard with analytics
- Role-based access control
- Clean and responsive UI

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js / Express
- Database: JSON / File-based storage
- Deployment: Replit / Local

## 📌 How It Works

- Users can register as Buyer or Seller
- Sellers can add products with pricing and stock
- Buyers can browse and place bulk orders
- Admin can monitor users, products, and orders

## 💡 Future Improvements

- Chat system between buyer and seller
- Email notifications
- Payment gateway integration
