# 🛒 B2B Trade Portal

A frontend web platform connecting buyers and sellers for bulk trading without middlemen.

---

## 🚀 Live Demo

👉 https://b2b-trade-portal.onrender.com

---

## ✨ Features

* 🔐 Login & Registration UI
* 🛍️ Marketplace UI
* 📦 Buyer & Seller dashboards (UI)
* 💬 Chat interface (UI)
* 📱 Responsive design

---

## 🧑‍💻 Tech Stack

* HTML
* CSS
* JavaScript
